## 🎁 AI-Based Gift Recommendation System

As part of our capstone project, we developed an **AI-powered gift recommendation system** designed to reduce the confusion and anxiety often associated with choosing gifts. The system allows users to input details about the recipient, which are then processed to generate **personalized product recommendations**.

---

## 🔍 Research & Analysis

The project began with an extensive **analysis phase**, focusing on understanding user needs and expectations. We conducted a **market research survey with over 130 participants**, which played a critical role in shaping both the system architecture and its core requirements.

### Key Insights

- 💰 **Budget range** is the most important constraint for users  
- 🎓 The primary target audience consists of **students**  
- 🤔 A major cause of indecision is **lack of knowledge about the recipient’s exact needs**

### Design Decisions

Based on these findings, we prioritized:

- Budget-aware filtering in the recommendation algorithm  
- Flexible input handling to account for uncertain or incomplete user knowledge  

---

## ⚙️ System Architecture

The system is built on a **two-step recommendation engine**:

### 1. Candidate Filtering
- Uses the **Amazon product dataset**  
- Narrows down a large pool of items  
- Applies constraints such as budget and general preferences  

### 2. Personalized Matching
- Matches products with the **user-provided recipient profile**  
- Uses metadata and product attributes to refine recommendations  

---

## 💡 Explainable Recommendations

To differentiate our system from traditional e-commerce filters, we implemented an **explanation generator**.

- 📊 **72% of users** reported higher trust when explanations are provided  
- Each recommendation includes a **clear rationale**

### Benefits

- Increases user trust  
- Improves transparency  
- Enhances decision confidence  

---

## 🔒 Privacy Considerations

We followed strict data privacy principles:

- ❌ No permanent storage of user input  
- ✅ Stateless recommendation processing
